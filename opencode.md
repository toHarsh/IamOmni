# OpenCode Instructions

Read and follow `.ai/protocol.md`.

Before working:

- Read `.ai/brief.md`.
- Read `.ai/handoff.md`.
- Read `.ai/session-log.md`.
- Read `.ai/changed-files.md`.

After every user/assistant exchange:

- Append a compact entry to `.ai/session-log.md`.
- Append transcript content to `.ai/transcripts/YYYY-MM-DD-opencode.md`.
- Update `.ai/changed-files.md` when files are created, updated, deleted, or heavily inspected.

Before switching tools or ending intentionally:

- Update `.ai/handoff.md`.
